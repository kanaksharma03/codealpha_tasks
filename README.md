# CodeAlpha Machine Learning Internship

This repository contains projects completed as part of the CodeAlpha Machine Learning Internship. The projects focus on applying machine learning techniques to solve real-world problems such as disease prediction, credit risk assessment, and handwritten text recognition.

## Projects

### 1. Disease Prediction Using Machine Learning

A machine learning system that predicts whether a patient is diabetic or non-diabetic based on medical parameters.

**Key Features**

* Data preprocessing
* Logistic Regression
* Random Forest Classifier
* Support Vector Machine (SVM)
* Model comparison
* Feature importance analysis
* New patient prediction
* Model saving and loading

---

### 2. Credit Scoring Model

A machine learning model developed to assess the creditworthiness of individuals based on financial and demographic attributes.

**Key Features**

* Data cleaning and preprocessing
* Credit risk classification
* Model training and evaluation
* Accuracy analysis
* Credit score prediction
* Performance comparison

---

### 3. Handwritten Text Recognition

A computer vision and machine learning project that recognizes handwritten characters or digits from images.

**Key Features**

* Image preprocessing
* Handwritten digit/character recognition
* Machine learning based classification
* Prediction of handwritten inputs
* Accuracy evaluation
